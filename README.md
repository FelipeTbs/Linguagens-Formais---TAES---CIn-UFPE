# Linguagens Formais - TAES CIn-UFPE
Repositório para amostragem do projeto de Especificações Formais do ciclo de Engenharia de Software. 
Disciplina de Tópicos Avançados em  Engenharia de Software. Ministrado pelo professor Augusto Cezar Alves Sampaio.

##Tema: Avaliação Empírica de LLMs Generalistas vs. Especializados na Síntese de Especificações Formais em Alloy

##Escopo do projeto: 

###Objetivo:

Comparar a performance de modelos de linguagem generalistas (ex: ChatGPT, Claude) e especializados (ex: CodeLlama, Phind) na geração automática de especificações Alloy a partir de descrições em linguagem natural.

###Metodologia
	•	Receber um conjunto pequeno (5–10) de problemas descritos em linguagem natural.
	•	Gerar modelos Alloy com diferentes LLMs usando prompts padronizados.
	•	Avaliar cada saída segundo: Correção sintática, Completude, Clareza.
